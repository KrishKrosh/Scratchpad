# Scratchpad

A lightweight SwiftUI macOS starter app.

## Requirements

- macOS
- Xcode 17+

## Run

1. Open `Scratchpad.xcodeproj` in Xcode.
2. Select the `Scratchpad` scheme.
3. Press `Cmd+R` to build and run.
